## 📌 Project Overview

The OLA Ride Booking Data Analysis Project is an end-to-end data analytics project that leverages SQL and Power BI to analyze ride booking data and generate actionable business insights.

The project focuses on understanding booking trends, customer behavior, ride cancellations, revenue generation, payment preferences, and driver performance. Through interactive dashboards and SQL-based analysis, the project helps transform raw ride data into meaningful business intelligence.



## 🎯 Project Objectives

* Analyze ride booking trends and patterns
* Understand customer and driver behavior
* Identify key reasons for ride cancellations
* Measure revenue generated from successful rides
* Evaluate customer and driver ratings
* Build interactive dashboards for business insights





## 🛠️ Tools & Technologies Used

* 🗄️ SQL – Data querying and business analysis
* 📊 Power BI – Dashboard creation and visualization
* 📑 Microsoft Excel – Data handling and exploration
* 📈 Data Analytics – Insight generation and reporting



## 🔍 SQL Analysis

The following business questions were solved using SQL:

### Booking Analysis

* Retrieve all successful bookings
* Calculate total customer cancellations
* Identify incomplete rides and their reasons

### Customer Analysis

* Find the top 5 customers with the highest number of bookings
* Calculate average customer ratings by vehicle type

### Driver Analysis

* Find maximum and minimum driver ratings for Prime Sedan rides
* Analyze driver cancellation reasons

### Revenue Analysis

* Calculate total booking value of successful rides
* Analyze payment method preferences including UPI

### Vehicle Analysis

* Calculate average ride distance by vehicle type

## 📈 Key Insights

* Successful rides generated the majority of total revenue.
* UPI emerged as one of the most preferred payment methods.
* Certain vehicle categories recorded higher average ride distances.
* Customer ratings highlighted service satisfaction trends.
* Cancellation analysis revealed operational improvement opportunities.
* A small group of customers contributed significantly to overall revenue.
